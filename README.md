# 🏁 Simulador de Corrida - Mario Kart (JavaScript)

Projeto desenvolvido em JavaScript simulando corridas entre personagens com base em atributos como velocidade, manobrabilidade e poder.

## 🚀 Funcionalidades

- 🎲 Sistema de rolagem de dados
- 🧠 Lógica de decisão baseada em tipos de pista:
  - RETA → velocidade
  - CURVA → manobrabilidade
  - CONFRONTO → poder
- 🥊 Sistema de confronto entre jogadores
- 🏆 Sistema de pontuação e vencedor

## 🛠️ Tecnologias

- JavaScript (Node.js)

## 📚 Aprendizados

- Estrutura de objetos
- Switch case
- Objetos
- Loops
- Funções assíncronas (async/await)
- Lógica de programação
- Simulação de eventos aleatórios
- Criação de projetos NODE
